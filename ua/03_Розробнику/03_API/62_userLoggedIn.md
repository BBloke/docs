### Виведення та повернення інформації про користувача, що авторизований (як менеджер чи веб-користувач) або ні
string userLoggedIn();

***

#### Формат даних результату:

	Array (
		[loggedIn] => 1
		[id] => 1
		[username] => siteadmin
		[usertype] => web
	)

***

#### Приклад:

	$info = $modx->userLoggedIn();
	//Поверне інформацію про користувача.
